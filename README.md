# store_gaming
Este é um projeto desenvolvido como parte da avaliação prática do curso de graduação em Ciência da Computação. O objetivo deste projeto é criar uma loja de jogos online que permita que os usuários naveguem, comprem e acessem jogos de maneira fácil e intuitiva.

## Detalhes do Projeto
- Nome do Projeto: Loja de Jogos
- Tipo: Aplicação Web
- Visão Geral: Os usuários poderão explorar uma variedade de jogos, ler resenhas e efetuar compras.

## Funcionalidades
- Navegação através de categorias de jogos
- Sistema de avaliação e comentários dos usuários
- Carrinho de compras e processamento de pagamentos
- Sistema de registro e login de usuários

## Tecnologias Utilizadas
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Ferramentas: Git, GitHub, Web Storm

## Requisitos dos Membros da Equipe
- Conhecimento em Desenvolvimento Web (Frontend e Backend)
- Habilidade em trabalhar em equipe e colaborar em diferentes partes do projeto
- Comprometimento com os prazos e a qualidade do código

## Considerações Finais
Este projeto oferece uma excelente oportunidade para aplicar conhecimentos teóricos em um projeto prático, proporcionando experiência em desenvolvimento de software e trabalho em equipe.
